Apport théorique

Capsule 10 min

1 Les 4 règles UX d'un dashboard RH

1. Lisibilité : 5 à 7 KPI max par écran. 2. Hiérarchisation : les alertes en rouge en haut à gauche. 3. Cohérence : même palette de couleurs pour tous les états. 4. Contexte : chaque KPI doit afficher valeur + objectif + tendance.

2 Dashboard stratégique vs opérationnel

Le dashboard stratégique s'adresse à la direction (COMEX) : vision globale, tendances, alertes majeures. Le dashboard opérationnel s'adresse aux RH : détail, calculs, actions quotidiennes.

3 Structure type d'un dashboard RH

Vue 1 — Global : effectif, turnover, absentéisme, coût masse salariale. Vue 2 — Recrutement : pipeline, délai, coût. Vue 3 — Social : formation, accidents, satisfaction. Vue 4 — Alertes : KPI hors seuil avec action associée.

4 Erreurs classiques

Dashboard trop chargé (> 10 KPI), absence d'objectif de référence, couleurs incohérentes, pas de source ni de date de mise à jour, KPI sans propriétaire identifié.
