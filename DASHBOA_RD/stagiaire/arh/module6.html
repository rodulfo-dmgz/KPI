Apport théorique

Capsule 10 min

1 Structure d'un fichier Excel RH

Un fichier RH bien structuré comporte 3 onglets : Données brutes (jamais toucher), Calculs (formules et KPI), Dashboard (graphiques et résumé visuel). Cette séparation évite les erreurs et facilite les mises à jour.

2 Tableau croisé dynamique (TCD)

Le TCD est l'outil incontournable de l'ARH. Il permet de regrouper, filtrer et croiser les données RH en quelques clics. Exemple : effectif par service + par type de contrat + par tranche d'âge.

3 Graphiques adaptés aux KPI RH

KPI unique → Jauge ou carte KPI. Évolution dans le temps → Courbe. Comparaison → Histogramme. Répartition → Camembert (max 5 catégories). Corrélation → Nuage de points.

4 Logique BI (Power BI / Tableau)

Les outils BI connectent directement aux sources de données (SIRH, paie) et actualisent le dashboard automatiquement. L'ARH doit comprendre le modèle de données (tables, relations, agrégats) même sans programmer.
