# Guide de stockage local

Ce site utilise le `localStorage` pour mémoriser :
- Le thème (`rd-theme`)
- La session utilisateur (`rd_auth`)

Aucune donnée n'est transmise à un serveur. La démo utilise des identifiants factices.

Pour effacer toutes les données : ouvrez les outils développeur > Application > Local Storage > supprimer les clés.
